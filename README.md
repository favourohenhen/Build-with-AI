# Build With AI - Event Schedule

A lightweight, responsive front-end web application built to display the event schedule for the **Build With AI** conference (Innovating the Future with Google Technologies).

## 🚀 Features

- **Event Schedule**: View a comprehensive list of talks, speakers, and time slots.
- **Search & Filter**: Instantly search for talks by category using the built-in search bar.
- **Responsive Design**: A mobile-friendly layout that looks great on any device.
- **Expandable Descriptions**: Read more about each talk with collapsible description sections.
- **Loading & Empty States**: Clean UI feedback while schedule data is loading or when a search yields no results.

## 🛠️ Tech Stack

- **HTML5**: Semantic markup.
- **CSS3**: Custom properties (variables), Grid, and Flexbox for modern layout and styling.
- **Vanilla JavaScript**: Handles dynamic data rendering and client-side search filtering (via `app.js`).

## 🏁 Getting Started

This project uses vanilla web technologies and requires no build step.

1. Clone or download the repository.
2. Navigate to the `public/` directory.
3. Open `index.html` in your favorite modern web browser.
